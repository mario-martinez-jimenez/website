# Mario Martinez-Jimenez - GitHub Pages Site

This is a static website ready to publish on GitHub Pages.

## Files

- `index.html` - main page
- `styles.css` - styles
- `assets/profile.jpg` - profile photo
- `assets/Mario_Martinez_Jimenez_CV.pdf` - CV
- `.nojekyll` - ensures plain static serving on GitHub Pages

## Publish

## Option 1: User site (recommended)

1. Create a repository named `mario-martinez-jimenez.github.io`.
2. Put these files in the root.
3. Commit and push to `main`.
4. The site will be live at:
   `https://mario-martinez-jimenez.github.io/`

## Option 2: Project site

1. Push this folder to any repository.
2. In GitHub: `Settings` -> `Pages`.
3. Set `Build and deployment` to `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. The site URL will be:
   `https://mario-martinez-jimenez.github.io/<repo-name>/`
